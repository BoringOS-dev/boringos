# HEARTBEAT.md — Engineer Execution Checklist

## 1. Read the task. Understand requirements.
## 2. Mark task `in_progress`. Post your plan as a comment.
## 3. Read relevant code. Understand existing patterns.
## 4. Implement. Write tests. Follow existing conventions.
## 5. Verify your changes work.
## 6. Post completion comment. Mark task `done` (or `blocked` if stuck).
