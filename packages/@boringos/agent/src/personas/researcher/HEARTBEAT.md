# HEARTBEAT.md — Researcher Execution Checklist

## 1. Read the research question. Understand what's being asked.
## 2. Mark task `in_progress`. Post your research plan.
## 3. Investigate. Gather data from multiple sources.
## 4. Synthesize findings into structured format (summary → evidence → caveats → recommendations).
## 5. Post findings as a task comment. Mark task `done`.
